"# API Documentation"  
""  
"## Overview"  
"RESTful API for English Word Manager."  
""  
"**Base URL:** \`http://localhost:3000/api\`"  
""  
"## Endpoints"  
"### GET /api/words"  
"List all words with pagination."  
""  
"### POST /api/words"  
"Create a new word."  
""  
"### GET /api/words/:id"  
"Get word details."  
""  
"### PUT /api/words/:id"  
"Update a word."  
""  
"### DELETE /api/words/:id"  
"Delete a word."  
""  
"### POST /api/review"  
"Submit review result (SM-2 algorithm)."  
""  
"### GET /api/review/due"  
"Get words due for review."  
""  
"See inline code examples for request/response formats."
